# Core Java Basics

This repository contains basic programs written while learning Core Java.

## Topics Covered
- Access Modifiers
- Constructors
- Variables and Data Types
- Conditional Statements (if, switch)
- Loops (for, while, do-while)
- Methods
- OOP Concepts
  - Classes and Objects
  - Inheritance
  - Polymorphism
  - Abstraction
  - Encapsulation
- Exception Handling

## Tools Used
- Java
- IntelliJ IDEA
- Git & GitHub
